## Setup Instructions

1. Create a new GitHub repository named `econosave-legal`
2. Push these files to the repository
3. Enable GitHub Pages in repository settings:
   - Go to Settings > Pages
   - Source: Deploy from a branch
   - Branch: main (or master) / docs folder
   - Click Save
4. Your privacy policy will be live at: `https://yourusername.github.io/econosave-legal/privacy-policy.html`
